\subsection{Testing Execution Details}
All tests are configured under the Maven Surefire configuration to run isolated inside the JVM. 
The test logs are redirected to clean local files inside the \code{log/} folder:
\begin{itemize}
    \item \textbf{JVM Native Errors}: Redirected to \code{log/hs\_err\_pid*.log}.
    \item \textbf{Surefire Replay Logs}: Redirected to \code{log/replay\_pid*.log}.
    \item \textbf{General System Output}: Redirected to \code{log/app.log}.
\end{itemize}
This ensures that the project root directory remains clean and uncluttered.
